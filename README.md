# mySaasProject
## Как это же работает?
Нужно запускать через `docker compose up`, в docker-compose.yml есть нужные сервисы(контейнер), 
хочешь запустить отдельно 1 сервис(контейнер) `docker compose "имя сервиса"`

# Приложения
1. Мониторинг курса криптовалют
2. Веб парсинг криптовалютных новостей(в процессе)
3. Мониториг курса национальных валют(в процессе)
